# IT Helpdesk Ticket Management System

A Python-based console application that simulates an IT Helpdesk Ticket Management System for logging, categorizing, prioritizing, and resolving hardware, software, and network issues.

This project is designed to reflect real-world Level-1 IT support and service desk operations.

## Features
- Create new IT support tickets
- Categorize issues (Hardware / Software / Network)
- Assign priority levels (Low / Medium / High)
- View all existing tickets
- Close resolved tickets
- File-based ticket storage for documentation

## Technologies Used
- Python 3
- File Handling
- Date & Time module

## How to Run
1. Install Python 3 on your system
2. Download or clone this repository
3. Run the script using:python ticket_system.py
4. Use the on-screen menu to create, view, and close tickets

## Use Case
This project simulates how IT support teams handle incident management by:
- Logging user-reported issues
- Classifying and prioritizing incidents
- Tracking ticket status
- Maintaining proper documentation
