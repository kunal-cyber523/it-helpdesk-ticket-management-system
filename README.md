# IT Helpdesk Ticket Management System

A Python-based console application that simulates an IT Helpdesk Ticket Management System for logging, categorizing, prioritizing, and resolving hardware, software, and network issues.

## Features
- Create new support tickets
- Categorize issues (Hardware / Software / Network)
- Assign priority levels (Low / Medium / High)
- View all tickets
- Close resolved tickets
- File-based ticket storage (data is saved across sessions)

## Technologies Used
- Python 3
- File Handling
- Date & Time module

## How to Run
1. Install Python 3 on your system
2. Run the script `ticket_system.py`
3. Follow the on-screen menu to create, view, and close tickets

## Learning Outcomes
- Learned incident management workflow in IT support
- Practiced file handling and console-based Python programming
- Understood ticket lifecycle: creation → prioritization → closure
